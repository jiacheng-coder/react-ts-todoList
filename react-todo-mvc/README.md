# Todo MVC

> React Hooks + Typescript + RooUI

## 安装

`npm i`

## 运行 

`npm run start`

## 使用指南

- 双击已添加的事项，进入编辑状态，页面失去焦点后自动同步数据
- 点击footer切换路由，可查看不同状态的事项（已完成和未完成）

## Hook

- useState
- useEffect
- useMemo
- useCallback
- useContext
